
Golden Peace Bar Image Assets

This zip file contains placeholder images for all bars in the Golden Peace app.
To use these assets:

1. Unzip this file
2. Move all the .imageset folders to your Xcode project's Assets.xcassets/BarImages folder
3. Replace the placeholder images with actual bar photos as you get them

The placeholder images are named according to the English bar names in your BarNameTranslation.swift file.
